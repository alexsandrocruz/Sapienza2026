## Business Intelligence Dashboard (DATA-001)
**Preço:** R$ 12.500  
**Prazo:** 4 semanas  
**Tipo:** Projeto

### O que é?
Painel visual ao vivo com seus dados (vendas, estoque, KPIs) em Power BI, Tableau ou Metabase. Clica e vê tudo.

### Como funciona?
1. Conectamos todas suas fontes de dados (ERP, Excel, banco de dados)
2. Limpamos e organizamos os dados
3. Criamos 30-50 gráficos interativos
4. Treinamos 4 horas seu time

### O que entregamos?
- Dashboard web e mobile
- 5 relatórios automáticos (WhatsApp/Email)
- Dicionário de dados
- 1 hora de treinamento mensal por 3 meses

### Para quem é ideal?
- Diretores que querem ver vendas em tempo real
- Gerentes confusos com Excel

### Benefícios
- Decisões 5x mais rápidas
- Zero erros de relatório
- Acesso mobile no celular

### Pergunta comum: "Vocês fazem dashboard?"
**Sim!** Criamos dashboards interativos em Power BI, Tableau ou Metabase conectados a todas suas fontes de dados. Investimento: R$ 12.500, pronto em 4 semanas.

## Data Lake e Data Warehouse (DATA-002)
**Preço:** R$ 19.500  
**Prazo:** 6-8 semanas  
**Tipo:** Projeto

### O que é?
O "Google" dos seus dados. Centraliza Excel, ERP, Google Sheets e bancos de dados em um único lugar, limpo e rápido.

### Como funciona?
1. Extraímos todos os dados
2. Organizamos em modelo estrela
3. Criamos pipelines automáticos
4. Governança + segurança

### O que entregamos?
- Data Lake na AWS ou GCP
- 20 tabelas prontas
- ETL diário automatizado
- Dicionário de dados
- Conformidade LGPD

### Para quem é ideal?
- Empresas com dados espalhados
- Relatórios demorados
- Analistas frustrados

### Benefícios
- Analytics 10x mais rápido
- Dados confiáveis
- Escala infinita

### Pergunta comum: "Precisamos centralizar nossos dados, vocês fazem?"
**Sim!** Implementamos Data Lake/Data Warehouse que centraliza todos os seus dados numa única fonte confiável. Investimento: R$ 19.500.
